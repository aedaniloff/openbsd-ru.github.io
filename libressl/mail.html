LibreSSL Почтовые рассылки

Для обсуждения LibreSSL есть две рассылки:

libressl@openbsd.org - открытая для всех, где обсуждаются технические стороны проекта LibreSSL на любой ОС.

libressl-security@openbsd.org - доступная только для разработчиков LibreSSL, где обсуждаются уязвимости в OpenSSL или LibreSSL.

Больше информации касательно рассылок можно найти на странице https://www.openbsd.org/mail.html.
