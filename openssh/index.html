Релиз OpenSSH 9.5 состоялся 4 октября 2023 года

OpenSSH это основной инструмент для создания соединений по протоколу SSH. Он шифрует весь трафик, что исключает прослушивание, захват соединений, и другие атаки. Кроме того, OpenSSH предоставляет широкие возможности для создания безопасного туннелирования, несколько методов аутентификации и множество раличных настроек.

OpenSSH состоит из следующих компонентов:

  • Операции с удаленными системами осуществляются при помощи ssh, scp, и sftp.
  • Управление ключами при помощи ssh-add, ssh-keysign, ssh-keyscan, и ssh-keygen.
  • Серверная часть состоит из sshd, sftp-server, и ssh-agent.

OpenSSH написан несколькими разработчиками проекта OpenBSD и распростаняется по лицензии BSD.

OpenSSH включен во многие коммерческие продукты, но очень не многие компании помогают OpenSSH с финансированием.

Финансовая помощь OpenSSH может быть оказана через OpenBSD Foundation.
