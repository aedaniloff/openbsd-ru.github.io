OpenSSH 7.9 вышла 19 октября 2018

OpenSSH это основной инструмент для соединения по протоколу SSH. Он шифруте весь траффик, чтобы исключить подслушивание, захват соединий, и другие атаки. Кроме того, OpenSSH предоставляет широкие возможности для создания безопасного туннелирования, несколько методов аутентификации и множество раличных настроек.

OpenSSH состоит из следующих компонентов:

  • Remote operations are done using ssh, scp, and sftp.
  • Key management with ssh-add, ssh-keysign, ssh-keyscan, and ssh-keygen.
  • The service side consists of sshd, sftp-server, and ssh-agent.

OpenSSH написан несколькими разработчиками OpenBSD Project и распростаняется по лицензии BSD.

OpenSSH включен во многие коммерческие продукты, но очень не многие компании помогают OpenSSH с финансированием.

Финансовая помощь OpenSSH может быть оказана через OpenBSD Foundation.
