OpenBGPD Зеркала

Релиз портированной версии OpenBGPD 6.6p0 состоялся первого ноябрая 2019 года.

Это порт OpenBGPD для других операционных систем. Он основан на коде портированных версий OpenNTPD, OpenSSH, и LibreSSL .

Требования к платформе

Портированная версия OpenBGPD собирается и работет на следующих платформах:

  • OpenBSD
  • Linux (Debian 9, Ubuntu 14.04)
  • FreeBSD (12.x)

OpenBGPD может работать и на других операционных системах, более новых или старых, но вышеперечисленные системы регулярно проверяются разработчиками. Отчеты (успех или нет) приветствуются. Вы можете сообщить об ошибках или отправить pull requests в GitHub project.

Цифровая подпись

OpenBGPD portable releases are signed with a signify(1) public key: RWTGJEPobKfXIP1+foh6O3ZGQIDIxwB0Hp4rH6fMYGdYPQMFH2IEqxcY

They are co-signed with GPG using this public key.

Release signatures can be verified using the signify program from within the OpenBSD operating system itself, using the portable version of signify, which is available with recent Debian, Ubuntu and Arch Linux releases, or by using GPG.
