OpenBGPD Документация

Web руководства доступны в самой OpenBSD. Эти руководства относятся к последней версии OpenBGPD.

  • bgpd - Border Gateway Protocol демон
  • bgpd.conf - Border Gateway Protocol конфигурационный файл демона
  • bgpctl - клиент для управления Border Gateway Protocol демоном
Протокол BGP, реализованный в OpenBGPD, описан в RFC 4271.
