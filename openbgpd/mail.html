OpenBGPD Почтовые рассылки

Основная рассылка пользователей OpenBGPD

Основная рассылка для обсуждений, вопросов, идей и предложений это users@openbgpd.org.

Чтобы подписаться на нее, просто отошли письмо на majordomo@openbgpd.org с текстом "subscribe users".

Для разработки портированной версии может использоваться любо Github, либо рассылка OpenBSD tech. О найденных ошибках можно сообщить либо в вышеупомянутый GitHub-проект, либо написать письмо (на английском естественно) в рассылку OpenBSD bugs.

Больше информации по этой теме можно найти на OpenBSD mailing list page.
