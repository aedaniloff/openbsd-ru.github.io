OpenSMTPD Документация (man-страницы)

Web руководства доступны в самой OpenBSD. Эти руководства относятся к последней версии OpenSMTPD.

  • smtpd(8) — демон SMTP (Simple Mail Transfer Protocol)
  • smtpd.conf(5) — конфигурационный файл SMTP демона
  • smtpctl(8) — менеджер SMTP демона
  • newaliases(8) — alias mappings генератор
  • makemap(8) — mappings генератор
  • aliases(5) — описание формата aliases файлов
  • forward(5) — формат forwarding файлов
  • table(5) — описание формата таблиц smtpd
  • sendmail(8) — просмотр очереди писем (a mail enqueuer) для smtpd

Протокол SMTP, реализованный в OpenSMTPD, описан в RFC 5321.
