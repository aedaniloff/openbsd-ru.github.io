OpenSMTPD Портированная версия

OpenSMTPD представляет из себя маленькую, безопасную и простую в использовании версию для проекта OpenBSD. Команда портирования добавляет в неё код, позволяющий ей работать на других операционных системах. Тем не менее, эта команда не занимается созданием и поддержкой пакетов для других ОС, поэтому пользователям, которые хотят использовать её на своих системах, мы рекомендуем связаться с нами.

OpenSMTPD работает на следующих UNIX-системах:

  • FreeBSD port
  • NetBSD / DragonFlyBSD
  • Linux (Archlinux package, Gentoo ebuild, Slackware, Debian, Fedora, Ubuntu)
  • Mac OS X

OpenSMTPD собирается на всех архитектурах, которые поддерживает OpenBSD.

Все комбинации систем / архитектур не были протестированны, но должны работать. Если же тут возникнут какие-то проблемы, сообщите нам - мы это исправим.

В основе портированной OpenSMTPD лежат исходники официальной версии, тем не менее их новые релизы выходят не синхронно. Портированные релизы по этой причине имеют в названии букву 'p'(например, 5.4.4p1). Официальная OpenBSD-версия никогда не использует 'p' в номере версии.
